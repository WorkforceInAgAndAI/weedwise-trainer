Images contained in the Full88_Weeds should be formatted in the following format for the appropriate plant type. This will help ensure code down the line can detect the images. 

-Seed images - seed_1.jpg
-Vegetation only images -veg_1.jpg
-Reproductive part images - repro_1.jpg
-Plant containing both vegetative and reproductive parts - plant_1.jpg
-Grass species should contain an additional set of images of the ligule (collar region) - ligu_1.jpg

Each plant folder should contain 2-4 images of each species at each of the designated stages.
